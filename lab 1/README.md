Контрольные вопросы Ис 25 22 Стрелов Григорий.

1 Что называется алгоритмом?
Алгоритм это конечная последовательность действий, направленных на решение определённой задачи или достижение цели.
Главное, чтобы алгоритм был понятным, определённым и выполнимым.

2. Какие свойства должен иметь алгоритм?
Алгоритм должен обладать следующими свойствами:
Понятность каждый шаг должен быть чётко описан.
Конечность  алгоритм должен завершаться после конечного числа шагов.
Результативность  выполнение алгоритма должно приводить к решению задачи.
Массовость  алгоритм должен подходить для решения всей группы однотипных задач.
Эффективность  алгоритм должен решать задачу с минимальными затратами времени и ресурсов.

3. В чём отличие линейного алгоритма от алгоритма с ветвлением?
Линейный алгоритм — выполняется последовательно, шаг за шагом, без условий и ответвлений.

Алгоритм с ветвлением — включает условные операторы, позволяющие выбрать разные пути выполнения в зависимости от условий.

4. Для чего используются циклы?
Циклы позволяют многократно повторять одни и те же действия, пока выполняется определённое условие.
Это экономит время и упрощает программу, когда одно и то же действие нужно выполнить много раз.

Пример: вывод чисел от 1 до 10 на экран можно сделать с помощью цикла, вместо того чтобы писать 10 отдельных команд.

5. Какие типы циклов существуют в программировании?
Основные типы циклов:

Цикл с предусловием выполняется, пока условие истинно.

Цикл с постусловием выполняется хотя бы один раз, затем проверяет условие.

Цикл с параметром повторяется заданное число раз.
